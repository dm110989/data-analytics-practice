# 📊 Шпаргалка: Сравнение средних значений

> **Урок 5 · Статистика** — Дисперсионный анализ (ANOVA) и поправки на множественные сравнения

---

## 🧭 Когда что использовать

| Ситуация | Метод |
|----------|-------|
| 2 группы | t-тест (см. урок 4) |
| 3+ группы, 1 фактор | **One-way ANOVA** |
| 3+ группы, 2+ фактора | **Two-way / N-way ANOVA** |
| Попарные сравнения после ANOVA | Тьюки / Геймс-Хоувелл / поправки Холма |

---

## 🔬 ANOVA — суть

**Гипотезы:**
- **H₀:** средние всех групп не различаются
- **H₁:** хотя бы одна пара средних различается

> ⚠️ ANOVA говорит «различия есть», но **не говорит где именно**. Для этого нужны попарные сравнения.

**Интуиция:** ANOVA сравнивает два источника разброса:

```
F = MSS_B / MSS_W
    ↑              ↑
  различия      шум внутри
 между групп     групп
```

---

## 🧮 Математика ANOVA

### Суммы квадратов

```
SST  =  SSB  +  SSW
(общая)  (межгрупп.)  (внутригрупп.)
```

| Величина | Формула | Степени свободы |
|----------|---------|-----------------|
| **SST** | Σ(Xᵢⱼ − X̄̄)² | N − 1 |
| **SSB** | Σ nᵢ(X̄ᵢ − X̄̄)² | m − 1 |
| **SSW** | Σ Σ(Xᵢⱼ − X̄ᵢ)² | N − m |

Где: `m` — кол-во групп, `N` — общий размер выборки, `nᵢ` — размер группы `i`

### Средние квадраты и F-статистика

```python
MSS_B = SSB / (m - 1)   # межгрупповой средний квадрат
MSS_W = SSW / (N - m)   # внутригрупповой средний квадрат

F = MSS_B / MSS_W
```

**p-value** считается по **F-распределению (распределению Фишера)** — оно асимметрично и строго ≥ 0.

## 🐍 ANOVA в Python

### scipy — минималистично

```python
import scipy.stats as st

st.f_oneway(group_red, group_green, group_blue)
# → F_onewayResult(statistic=85.99, pvalue=3.44e-30)
```

### statsmodels — гибко

```python
import statsmodels.formula.api as smf
from statsmodels.stats.api import anova_lm

model = smf.ols(formula="likes ~ button", data=df).fit()
anova_lm(model)
# p-value → столбец PR(>F)
# вверху SSB, внизу SSW (Residual)
```

> 💡 По сути это линейная регрессия!

### pingouin — золотая середина ✨

```python
import pingouin as pg

pg.anova(data=df, dv="likes", between="button")
# столбцы: ddof1, ddof2, F, p-unc, np2

# ANOVA Уэлча (при неравенстве дисперсий):
pg.welch_anova(data=df, dv="likes", between="button")
```

## ⚠️ Проблема множественных сравнений

При k попарных сравнениях с α = 0.05 вероятность хоть раз ошибиться (ошибка I рода) резко растёт:

```python
from scipy.special import comb

k = comb(N=4, k=2, exact=True)   # = 6 сравнений
p_all_correct = 0.95 ** k         # = 0.735
p_at_least_one_error = 1 - p_all_correct  # = 0.265 (!!)
```

> С 4 группами вероятность хоть раз ошибиться уже **~26%** вместо 5%.

## 🛡️ Поправки на множественные сравнения

| Поправка | Суть | Когда |
|----------|------|-------|
| **Бонферрони** | α' = α / k | Консервативная, ↑ ошибка II рода |
| **Холм-Бонферрони** | Поэтапная корректировка | Мягче Бонферрони |
| **Бенджамини-Хохберг** | Контроль FDR | Много сравнений |
| **Тьюки (HSD)** | Специально для ANOVA | Равные дисперсии |
| **Геймс-Хоувелл** | Тьюки + поправка на дисперсии | Неравные дисперсии |

### В pingouin

```python
# Попарные сравнения с поправкой Холма:
pg.pairwise_tests(data=df, dv="likes", between="button", padjust="holm")
# p-unc → до корректировки, p-corr → после

# Тьюки:
pg.pairwise_tukey(data=df, dv="likes", between="button")
# p-tukey, mean(A), mean(B), diff

# Геймс-Хоувелл:
pg.pairwise_gameshowell(data=df, dv="likes", between="button")
# pval
```

## 🔀 Многофакторный ANOVA

Когда результат зависит от **двух и более факторов** (цвет + структура страницы; тип рекламы + возраст).

### Суммы квадратов (two-way):

```
SST = SSW + SSB_A + SSB_B + SSB_A×B
                              ↑
                        взаимодействие
```

### Три нулевые гипотезы:

| Гипотеза | Смысл |
|----------|-------|
| **H₀A** | Средние по фактору A не различаются |
| **H₀B** | Средние по фактору B не различаются |
| **H₀AB** | Факторы A и B независимы (взаимодействия нет) |

### В Python

```python
# statsmodels:
formula = "clicks ~ ads * age_group"
# что равно: "clicks ~ ads + age_group + ads:age_group"
model = smf.ols(formula=formula, data=df).fit()
anova_lm(model)

# pingouin:
pg.anova(data=df, dv="clicks", between=["ads", "age_group"])
```

## 📈 Визуализация взаимодействия

Строятся **interaction plots** — графики средних для каждой комбинации факторов:

| Вид графика | Интерпретация |
|-------------|---------------|
| Линии **параллельны** | Взаимодействия нет |
| Линии **непараллельны** | Есть взаимодействие |
| Линии **пересекаются** | Сильное взаимодействие (эффект меняет знак) |

> ⚠️ Отсутствие видимого взаимодействия на глаз ≠ статистическая незначимость. Всегда смотрите на p-value.

---

## 📦 Меры величины эффекта

| Метрика | Где | Смысл |
|---------|-----|-------|
| **η² (эта-квадрат)** | ANOVA | Доля дисперсии, объяснённой фактором |
| **partial η² (np2)** | pingouin по умолчанию | Как η², но относительно SSW + SSB |
| **ω² (омега-квадрат)** | Скорректированный | Менее смещённый, чем η² |
| **d Коэна / g Хеджеса** | Попарные сравнения | Различие в единицах σ |

## 📋 Алгоритм на практике

```
1. Проверить допущения (нормальность, равенство дисперсий)
   └─ нарушены? → ANOVA Уэлча / непараметрические тесты

2. Провести ANOVA
   └─ p > α? → Различий нет, стоп.
   └─ p ≤ α? → Различия есть, идём дальше.

3. Попарные сравнения с поправкой
   ├─ Равные дисперсии → Тьюки
   └─ Неравные дисперсии → Геймс-Хоувелл

4. Оценить величину эффекта (np2, η²)

5. Визуализировать результаты
```

## 🔗 Быстрый выбор поправки

```
Дисперсии равны?
    ├─ Да → Тьюки (pairwise_tukey)
    └─ Нет → Геймс-Хоувелл (pairwise_gameshowell)

Много гипотез, контроль FDR важен?
    └─ Бенджамини-Хохберг (padjust="fdr_bh")

Строгий контроль ошибки I рода любой ценой?
    └─ Бонферрони (padjust="bonf") — но осторожно с ошибкой II рода
```
