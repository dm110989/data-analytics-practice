# Статистика: шпаргалка для аналитика данных

```python
import numpy as np
import pandas as pd
from scipy import stats
import statsmodels.api as sm
```

---

# 1. Описательная статистика

Описательная статистика — это способ **кратко описать данные**: что в центре, насколько данные разбросаны, есть ли выбросы.

---

## Меры центра

### Среднее (Mean)

Сумма всех значений, делённая на их количество.

```python
df["salary"].mean()
np.mean([10, 20, 30])  # → 20.0
```

⚠️ Чувствительно к выбросам: один миллионер сильно поднимет среднее.

---

### Медиана (Median)

Значение посередине, если отсортировать данные. Устойчива к выбросам.

```python
df["salary"].median()
np.median([10, 20, 30, 1000])  # → 25.0
```

> Если среднее сильно выше медианы — скорее всего, есть выбросы сверху.

---

### Мода (Mode)

Самое частое значение.

```python
df["city"].mode()[0]
stats.mode([1, 2, 2, 3])  # → 2
```

---

## Меры разброса

### Дисперсия (Variance)

Средний квадрат отклонений от среднего. Показывает, насколько данные «разбросаны».

```python
df["salary"].var()    # выборочная (делится на n-1)
np.var([10, 20, 30])  # генеральная (делится на n)
```

---

### Стандартное отклонение (Std)

Корень из дисперсии. Удобнее, потому что в тех же единицах, что и данные.

```python
df["salary"].std()
np.std([10, 20, 30], ddof=1)  # ddof=1 — выборочное
```

> Правило «трёх сигм»: ~68% данных попадают в диапазон mean ± std, ~95% в mean ± 2*std.

---

### Квантили и перцентили

Квантиль — значение, ниже которого находится заданная доля данных.

```python
df["salary"].quantile(0.25)  # 25-й перцентиль (Q1)
df["salary"].quantile(0.50)  # медиана (Q2)
df["salary"].quantile(0.75)  # 75-й перцентиль (Q3)
df["salary"].quantile([0.25, 0.5, 0.75])  # сразу несколько
```

---

### IQR — межквартильный размах

Разница между Q3 и Q1. Показывает разброс «средних» 50% данных.

```python
Q1 = df["salary"].quantile(0.25)
Q3 = df["salary"].quantile(0.75)
IQR = Q3 - Q1
```

Используется для поиска выбросов:

```python
lower = Q1 - 1.5 * IQR
upper = Q3 + 1.5 * IQR
outliers = df[(df["salary"] < lower) | (df["salary"] > upper)]
```

---

## Форма распределения

### Асимметрия (Skewness)

Показывает, «перекошено» ли распределение.

```python
df["salary"].skew()
```

- `skew ≈ 0` — симметрично
- `skew > 0` — хвост вправо (большинство значений слева, есть выбросы справа)
- `skew < 0` — хвост влево

---

### Эксцесс (Kurtosis)

Насколько острый «пик» у распределения.

```python
df["salary"].kurt()
```

- `kurt ≈ 0` — нормальное распределение
- `kurt > 0` — острый пик, много выбросов
- `kurt < 0` — плоское, мало выбросов

---

## Быстрый осмотр

```python
df.describe()
# count, mean, std, min, 25%, 50%, 75%, max
```

---

# 2. Распределения

Распределение — это описание того, как часто встречаются разные значения.

---

## Нормальное распределение (Гауссово)

Симметричное колоколообразное распределение. Основа большинства статистических методов.

```python
# Генерация
data = np.random.normal(loc=0, scale=1, size=1000)
# loc — среднее, scale — стандартное отклонение

# Проверка на нормальность
stat, p = stats.shapiro(data)
print(f"p-value = {p:.4f}")
# если p > 0.05 — распределение нормальное
```

> Обозначение: N(μ, σ²), где μ — среднее, σ — стандартное отклонение.

---

## Стандартное нормальное распределение

Нормальное с μ=0 и σ=1. Любую нормальную переменную можно привести к нему через **стандартизацию**.

```python
# z-оценка (z-score)
z = (x - mean) / std

# В pandas:
df["salary_z"] = (df["salary"] - df["salary"].mean()) / df["salary"].std()

# Через scipy:
z_scores = stats.zscore(df["salary"])
```

Z-score показывает, на сколько стандартных отклонений значение отличается от среднего.

---

## Равномерное распределение

Все значения одинаково вероятны.

```python
data = np.random.uniform(low=0, high=10, size=1000)
```

---

## Биномиальное распределение

Количество «успехов» в n независимых испытаниях с вероятностью успеха p.

```python
# n=10 бросков монеты, p=0.5 — вероятность орла
data = np.random.binomial(n=10, p=0.5, size=1000)

# Вероятность ровно k=3 успехов:
stats.binom.pmf(k=3, n=10, p=0.5)

# Вероятность k ≤ 3:
stats.binom.cdf(k=3, n=10, p=0.5)
```

---

## Распределение Пуассона

Количество событий за фиксированный промежуток времени (редкие события).

```python
# λ (lambda) — среднее количество событий
data = np.random.poisson(lam=5, size=1000)

# Вероятность ровно k=3 событий:
stats.poisson.pmf(k=3, mu=5)
```

---

## Центральная предельная теорема (ЦПТ)

Если брать достаточно большие выборки (n ≥ 30) из **любого** распределения, среднее этих выборок будет распределено нормально.

```python
# Проверка на примере:
means = [np.mean(np.random.exponential(scale=2, size=30)) for _ in range(1000)]
# means будут распределены нормально, даже если исходные данные — экспоненциальные
```

> Именно ЦПТ позволяет применять нормальные методы там, где данные не нормальны.

---

# 3. Проверка гипотез

Проверка гипотез — это статистический способ ответить на вопрос: «Случайна ли разница, которую я вижу, или она реальная?»

---

## Ключевые понятия

**H₀ (нулевая гипотеза)** — «разницы нет», «всё случайно».

**H₁ (альтернативная гипотеза)** — «разница есть», «есть эффект».

**p-value** — вероятность получить такой результат, если H₀ верна.

```text
p-value < 0.05 → отвергаем H₀ (разница статистически значима)
p-value ≥ 0.05 → не отвергаем H₀ (разница может быть случайной)
```

⚠️ p-value не говорит, насколько велика разница — только насколько она случайна.

**Уровень значимости (α)** — порог, обычно 0.05 (5%).

---

## t-тест

Сравнивает средние.

### Одновыборочный t-тест

Среднее выборки vs заданное значение.

```python
# Зарплата в компании ≠ 100?
stat, p = stats.ttest_1samp(df["salary"], popmean=100)
print(f"p-value = {p:.4f}")
```

---

### Двухвыборочный t-тест (независимые группы)

Сравнить средние двух групп.

```python
group_a = df[df["group"] == "A"]["salary"]
group_b = df[df["group"] == "B"]["salary"]

stat, p = stats.ttest_ind(group_a, group_b)
print(f"p-value = {p:.4f}")
```

---

### Парный t-тест

Одни и те же объекты до и после.

```python
stat, p = stats.ttest_rel(df["before"], df["after"])
print(f"p-value = {p:.4f}")
```

---

## Тест Манна-Уитни (непараметрический)

Аналог двухвыборочного t-теста, когда данные не нормальные.

```python
stat, p = stats.mannwhitneyu(group_a, group_b, alternative="two-sided")
print(f"p-value = {p:.4f}")
```

---

## Хи-квадрат тест (χ²)

Проверяет связь между двумя **категориальными** переменными.

```python
# Таблица сопряжённости
contingency = pd.crosstab(df["city"], df["product"])

stat, p, dof, expected = stats.chi2_contingency(contingency)
print(f"p-value = {p:.4f}")
```

---

## Тест Шапиро-Уилка

Проверка на нормальность распределения.

```python
stat, p = stats.shapiro(df["salary"])
# p > 0.05 → распределение нормальное
```

---

## ANOVA (дисперсионный анализ)

Сравнение средних трёх и более групп.

```python
g1 = df[df["city"] == "Moscow"]["salary"]
g2 = df[df["city"] == "SPB"]["salary"]
g3 = df[df["city"] == "Kazan"]["salary"]

stat, p = stats.f_oneway(g1, g2, g3)
print(f"p-value = {p:.4f}")
```

> Если p < 0.05 — хотя бы одна группа отличается. Чтобы узнать какая — нужен Post-hoc тест.

---

## Доверительный интервал

Диапазон значений, в котором с заданной вероятностью (обычно 95%) находится истинное среднее генеральной совокупности.

```python
import scipy.stats as stats

mean = df["salary"].mean()
n = len(df["salary"])
se = stats.sem(df["salary"])  # стандартная ошибка среднего

ci = stats.t.interval(0.95, df=n-1, loc=mean, scale=se)
print(f"95% ДИ: {ci[0]:.1f} — {ci[1]:.1f}")
```

> «95% ДИ: 90–110» означает: мы на 95% уверены, что истинное среднее между 90 и 110.

# 4. Корреляция и регрессия

---

## Корреляция

Показывает, есть ли **линейная связь** между двумя переменными и насколько она сильна.

### Коэффициент Пирсона (r)

Для числовых данных с нормальным распределением.

```python
r, p = stats.pearsonr(df["age"], df["salary"])
print(f"r = {r:.3f}, p = {p:.4f}")

# Матрица корреляций:
df.corr()
df[["age", "salary", "experience"]].corr()
```

Интерпретация r:

| r | Сила связи |
|---|---|
| 0.0 – 0.2 | нет связи |
| 0.2 – 0.4 | слабая |
| 0.4 – 0.6 | умеренная |
| 0.6 – 0.8 | сильная |
| 0.8 – 1.0 | очень сильная |

> r > 0 — прямая связь (растёт вместе). r < 0 — обратная (один растёт — другой падает).

---

### Коэффициент Спирмена (ρ)

Когда данные не нормальные или порядковые.

```python
r, p = stats.spearmanr(df["age"], df["salary"])
print(f"ρ = {r:.3f}, p = {p:.4f}")
```

---

### ⚠️ Корреляция ≠ Причинность

Высокая корреляция не означает, что A вызывает B. Оба могут зависеть от третьего фактора.

---

## Простая линейная регрессия

Моделирует зависимость одной переменной от другой: `y = a + b*x`.

- `b` — наклон: на сколько изменится y, если x вырастет на 1
- `a` — свободный член: y при x = 0

```python
# Через scipy
slope, intercept, r, p, se = stats.linregress(df["experience"], df["salary"])
print(f"y = {intercept:.1f} + {slope:.1f} * x")
print(f"R² = {r**2:.3f}")
```

```python
# Через statsmodels (больше деталей)
X = sm.add_constant(df["experience"])  # добавить столбец с 1 для intercept
model = sm.OLS(df["salary"], X).fit()
print(model.summary())
```

---

## Множественная линейная регрессия

Несколько предикторов: `y = a + b₁x₁ + b₂x₂ + ...`

```python
X = df[["experience", "age", "education"]]
X = sm.add_constant(X)
y = df["salary"]

model = sm.OLS(y, X).fit()
print(model.summary())
```

Что смотреть в `summary()`:

| Показатель | Что значит |
|---|---|
| `R²` | доля объяснённой дисперсии (0–1, чем выше — тем лучше) |
| `Adj. R²` | R² с поправкой на кол-во переменных |
| `coef` | коэффициент при переменной |
| `P>|t|` | p-value для каждого коэффициента |
| `[0.025, 0.975]` | 95% доверительный интервал коэффициента |

```python
# Предсказание
new_data = sm.add_constant(pd.DataFrame({"experience": [5], "age": [30], "education": [1]}))
model.predict(new_data)
```

---

## R² — коэффициент детерминации

Показывает, какую долю разброса y объясняет модель.

```python
model.rsquared        # R²
model.rsquared_adj    # Adjusted R²
```

- `R² = 0.8` → модель объясняет 80% изменчивости y
- `R² = 0.1` → модель объясняет только 10% → плохая

# Быстрая шпаргалка

| Задача | Метод | Код |
|---|---|---|
| Среднее | Mean | `df["col"].mean()` |
| Середина | Медиана | `df["col"].median()` |
| Разброс | Std, IQR | `df["col"].std()` |
| Нормальность? | Шапиро-Уилк | `stats.shapiro(data)` |
| Сравнить 2 средних | t-тест | `stats.ttest_ind(a, b)` |
| Сравнить 2 средних (не норм.) | Манн-Уитни | `stats.mannwhitneyu(a, b)` |
| Сравнить 3+ средних | ANOVA | `stats.f_oneway(g1, g2, g3)` |
| Связь категорий | Хи-квадрат | `stats.chi2_contingency(table)` |
| Линейная связь (норм.) | Пирсон | `stats.pearsonr(x, y)` |
| Линейная связь (не норм.) | Спирмен | `stats.spearmanr(x, y)` |
| Предсказать y через x | Регрессия | `sm.OLS(y, X).fit()` |

# Алгоритм анализа данных

```python
# 1. Описательная статистика
df.describe()
df["salary"].skew()

# 2. Проверить нормальность
stats.shapiro(df["salary"])

# 3. Найти выбросы
Q1, Q3 = df["salary"].quantile([0.25, 0.75])
IQR = Q3 - Q1

stats.mannwhitneyu(group_a, group_b) # если не нормальное

# 5. Корреляция
df.corr()
stats.pearsonr(df["x"], df["y"])

# 6. Регрессия
X = sm.add_constant(df[["x1", "x2"]])
model = sm.OLS(df["y"], X).fit()
print(model.summary())
```
